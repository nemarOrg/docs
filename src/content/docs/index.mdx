## What is NEMAR?

NEMAR (Neuroelectromagnetic Data Archive and Tools Resource) is a platform for archiving,
sharing, and processing neuroimaging datasets in Brain Imaging Data Structure (BIDS) format.
It is several systems working together; the command-line interface is the entry point most
users start with, but it is only one part.

## NEMAR surfaces at a glance

| Surface | URL | What it is |
| --- | --- | --- |
| CLI | `nemar` | Command-line tool for dataset management |
| API | `api.nemar.org` | Backend API (Cloudflare Workers + D1) |
| Data plane | `data.nemar.org` | Public dataset files, manifests, archive zips |
| Browser | `ww2.nemar.org` | Current dataset browser (Astro) |
| Viewer | `zarr.nemar.org` | In-browser signal viewer (Zarr proxy) |
| Docs | `docs.nemar.org` | This site |
