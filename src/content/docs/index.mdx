## What is NEMAR?

NEMAR (Neuroelectromagnetic Data Archive and Tools Resource) is a curated repository for
neurophysiology data in Brain Imaging Data Structure (BIDS) format. This CLI provides tools
for uploading, validating, versioning, and downloading datasets, with automatic git-annex,
S3, GitHub, and DOI integration.

## NEMAR at a glance

| Surface | URL | What it is |
| --- | --- | --- |
| API | `api.nemar.org` | Backend API (Cloudflare Workers + D1) the CLI talks to |
| Data plane | `data.nemar.org` | Public dataset files, manifests, and archive zips |
| Browser | `ww2.nemar.org` | Current dataset browser (Astro) |
| Viewer | `zarr.nemar.org` | In-browser signal viewer (Zarr proxy) |
| Docs | `docs.nemar.org` | This site |
